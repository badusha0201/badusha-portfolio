# A. Badusha - Premium Portfolio Website

A modern, premium portfolio website for A. Badusha showcasing software development, UI/UX design, AI/Data Science expertise, and business strategy services.

## ✨ Features

- **Modern Design**: Glassmorphism design with gradient backgrounds and dark theme
- **Smooth Animations**: Framer Motion for elegant transitions and micro-interactions
- **Fully Responsive**: Optimized for desktop, tablet, and mobile devices
- **Particle Effects**: Interactive particle background with connection lines
- **Dark/Light Mode**: Toggle between themes with smooth transitions
- **Performance Optimized**: Lazy loading and optimized assets for fast loading
- **Accessibility**: WCAG compliant with semantic HTML and ARIA labels
- **SEO Optimized**: Meta tags and structured data for search engines

## 🎯 Sections

1. **Hero** - Introduction with typing animation and CTA buttons
2. **About** - Biography, career goals, and key statistics
3. **Skills** - Animated progress bars showing expertise levels
4. **Projects** - Showcase of featured projects with tech stack
5. **Experience** - Timeline of professional experience and education
6. **Services** - Six core service offerings
7. **Testimonials** - Client testimonials with carousel navigation
8. **Blog** - Latest articles with category filtering
9. **Contact** - Contact form with validation and social links
10. **Footer** - Quick links and social media icons

## 🛠️ Tech Stack

- **Framer Motion** - Animation library
- **TypeScript** - Type safety
- **React Hook Form** - Form handling
- **React Icons** - Icon library

## 📦 Installation

1. Clone the repository
2. Install dependencies:
   ```bash
   npm install
   ```

3. Run the development server:
   ```bash
   npm run dev
   ```

4. Open [http://localhost:3000](http://localhost:3000) in your browser

## 🚀 Development

- `npm run dev` - Start development server
- `npm run build` - Build for production
- `npm start` - Start production server
- `npm run lint` - Run ESLint

## 📱 Responsive Design

The website is fully responsive and works seamlessly on:
- Desktop (1920px and above)
- Tablet (768px - 1024px)
- Mobile (320px - 767px)

## 🎨 Customization

### Colors
Edit the color palette in your styles configuration (e.g. design tokens or CSS variables):
- Primary: Blue (#3b82f6)
- Secondary: Purple (#a855f7)
- Accent: Pink (#ec4899)

### Content
Update portfolio content in:
- `utils/constants.ts` - Skills, projects, experience, etc.
- Component files - Section-specific content

### Animations
Modify animation timings in:
- `utils/animations.ts` - Animation variants
- Component files - Animation configurations
